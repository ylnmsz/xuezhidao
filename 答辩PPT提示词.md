## 项目背景

**项目名称：** 基于 Web 的题库练习与错题本系统（学知道）

**项目简介：** 一个面向K-12教育的智能化学习平台，支持题库管理、在线练习、作业布置批改、错题本、学情分析等功能。教师端可上传和管理题目（支持 CSV/DOCX 批量导入 + AI 解析），学生端可完成作业、自主练习、查看错题本和个人学情数据。

**技术栈：**
- 前端：Vue 3（Composition API）+ Pinia + Vue Router + Vite
- 后端：Node.js + Express
- 数据库：PostgreSQL
- 数学公式渲染：KaTeX
- AI 文档解析：DeepSeek API + MinerU OCR
- 认证：JWT（JSON Web Token）
- 图片处理：Cropper.js

### 第11页：系统特色与难点攻克

**标题：** 系统特色与难点攻克

**内容要点：**

**特色功能：**
- AI 智能解析 DOCX 文档（支持数学公式、表格、图片）
- 14+ 题型全覆盖
- 异步任务系统（页面跳转不中断）
- 积分/等级/战力激励体系
- 沉浸式练习模式

**攻克的技术难点：**
1. **DOCX 中数学公式丢失问题**
   - 原因：WMF 矢量图格式 MinerU 无法渲染
   - 解决：LibreOffice WMF→PNG 转换 + 快速 ZIP 预检
2. **AI 解析题型误判**
   - 如：几何命题题（①②③④）被误判为"阅读理解"
   - 解决：提示词工程 + 后处理规则 + 命题标号检测
3. **批量解析中断问题**
   - 原因：页面跳转中断 HTTP 请求
   - 解决：服务端异步任务 + 前端轮询 + sessionStorage 持久化
4. **数学问答题被误判为填空题**
   - 解决：多步后处理兜底 + 提示词强化

**配图建议：** 每个难点配一张"Before vs After"对比图（错误输出 vs 正确输出）。
